# 📟 OS Shell Sim | Operating System Emulator

A retro-futuristic terminal-based emulator that simulates core Operating System behaviors and scheduling algorithms.

## ✨ Features
- **Hacker Aesthetic**: Retro CRT scanline effects and green monochrome typography.
- **Process Scheduling**: Simulation of First Come First Served (FCFS) algorithm execution.
- **Command Architecture**: Supports standard terminal commands like `ls`, `clear`, and `help`.
- **System Internals**: Mock file system structure visualization.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML/CSS.
- **Logic**: Custom Command Parser in JavaScript.
- **Design**: CSS Grid and CRT Scanline filters.

## 🧠 Why I built this?
To simplify complex Operating System concepts. This project demonstrates my understanding of kernel-level abstractions, command-line interfaces, and process life-cycles.
